# 模板语料

创建或优化简历时读取本文件。

## 唯一模板

只使用：

`template/.程序员简历模板互联网IT行业程序员自动发货/后端简历/java简历/java简历/优秀简历汇总/xxx.doc`

这是蓝金风格 JavaEE 软件工程师简历模板。其他模板已经删除，不得从其他文件补样式。

## 版式规则

- 顶部使用居中标题，标题格式参考 `JavaEE软件工程师_3年_毛向军`。
- 标题下方使用蓝色主横条和金色尾段横条。
- 每个 section 使用蓝色标签条加右侧蓝色细线。
- 基本信息和求职意向使用双栏 key-value 排列。
- 开发技能使用箭头列表。
- 项目/经历使用项目名行 + 开发环境/技术栈 + 职责 bullet + 概述 bullet 的紧凑节奏。
- 缺失信息必须省略，不得保留 `Xxx`、电话样例、学校样例或目标城市样例。

## 内容规则

模板只提供样式和章节节奏。模板中的姓名、电话、公司、学校、日期、项目和技术栈都是样例，不是用户事实。
